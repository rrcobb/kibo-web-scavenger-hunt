DevTools Scavenger Hunt

Introduction

The DevTools are like a Detective's magnifying glass, notebook, and recorder all in one. Web Developers use the DevTools every day to look 'under the hood' of a webpage, see what's going on, and make experimental changes.

In this challenge, you'll use the DevTools to find messages hidden in the HTML and CSS of this page.

The DevTools

This page contains hidden messages. Follow the clues to put the pieces together.

You'll need to use the developer tools to find each message.

Open the developer tools by right clicking anywhere on the page and selecting 'Inspect' from the menu. You should see a panel open at the bottom or side of the page.

Great! You got the DevTools open, now let's take a tour.

The Elements panel shows the HTML elements that make up the page. You can see what's inside an element by clicking open the arrow on the left.

The Styles panel shows the CSS styles for an element. Click on an element in the Elements panel and look in the Styles panel to see all of the styles.

The Styles panel has a bonus power: you can click on the value for any property and change it! You can change the color of the text, or the font size, or the background color, or any other CSS property.

Tip: You can drag the panels to resize them. There's also a ton more panels that you can look at if you want to explore (check out what happens when you open the Network panel, then reload the page!)

Got it. What's next?

Instructions

You'll use the Elements panel to find HTML elements and inspect their attributes. You'll use the Styles panel to see CSS properties. By the end of the challenge, you'll make changes to the styles of the page in order to reveal the most deeply hidden messages.

Click the button below to get started. For each prompt:

  1. Read the clue and use the DevTools to find the answer.
  2. Type the answer in the box.
  3. Click "Check" to check your answer and get the next prompt.

You can check as many times as you want. You'll get feedback on whether your answer was right or wrong.

After you solve all the challenges, you'll uncover the final answer.

Challenge 1

The first secret message is: Climb Kibo. Type it into the box below.

This one's pretty easy. It's supposed to get you warmed up for solving the next few challenges.

Challenge 2

To find the second secret message, find the page title in the Elements panel. The title tag has the secret in it! In the Elements tab of the panel, click to open the head tag, and look for the title tag. The title is inside the head.

Challenge 3

The third secret is the name of this HTML element. Right click on the bold text and click 'Inspect'. The text 'this HTML element' is inside of a tag. The tag has angle brackets (< and >) and a name. That name is the third secret.

Challenge 4

The fourth secret is the value of an input element hidden inside this very prompt. Click to inspect an element in this paragraph, open up the elements and look nearby for an input element. The value attribute of the input element is the fourth secret.

Challenge 5

The fifth secret is hidden in the Styles panel. First, select the h1 element - it's the heading of the page, at the top of the <body>, and inside it says 'DevTools Scavenger Hunt'. Click that element in the Elements panel so that the Styles panel shows all the CSS properties and values. The secret is in the content property.

Challenge 6

To find the last value, you'll have to change a style using the developer tools. There's an image hidden off screen that has the secret.

Find the <img> tag by looking near this prompt in the Elements panel. Click on the <img> in the Elements tab to select it.

In the Styles tab, change the value for the 'left' property to move the element onto the screen. You can try different values, there's lots of ones that will work.

Congratulations!
