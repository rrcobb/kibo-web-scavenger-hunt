DevTools Scavenger Hunt

Introduction

The DevTools are like a Detective's magnifying glass, notebook, and recorder all in one. Web Developers use the DevTools every day to look 'under the hood' of a webpage, see what's going on, and make experimental changes.

In this challenge, you'll use the DevTools to find messages hidden in the HTML and CSS of this page.

The DevTools

You'll need to use the developer tools to find the messages hidden on this page.

To open the developer tools: right click anywhere on the page and select 'Inspect' from the menu. You should see a panel open at the bottom or side of the page.

(If you don't have a 'right click', try holding control and clicking to bring up the menu before clicking 'Inspect').

Tour of the devtools

The Elements panel shows the HTML elements that make up the page. You can open an element to see what's inside by clicking the arrow on the left. Try it in the Elements panel!

The Styles panel shows the CSS styles for an element. Click on an element in the Elements panel to see its styles in the Styles panel. (In this image, the <body> element is selected.)

The Styles panel has a bonus power: you can change the text color, the font size, the background color, or any other CSS property!

Try changing the 'color' of the body text from '#262626' to '#888' to see! You can change it back if that makes it hard to read.

One of the great things about the DevTools is the freedom to experiment. Nothing you change will break the site permanently. You can always refresh the page to get back to how things started

Challenge 1

The first secret message is in Elements panel in the <title> inside the <head>. The <title> tag has the secret in it!

In the Elements panel, click to open the <head> tag, and look for the <title> tag inside.

Type the secret into the box below.

Challenge 2

The second secret is hidden in an <input> element in the challenge 2 <div>.

Inspect an element within Challenge 2, like this sentence. Find this section in the Elements panel, and expand it using the arrow on the left.

Look nearby for an <input> element.

The value attribute of the input element contains the second secret.

Challenge 3

The third secret is the name of the HTML tag for this bold text.

Right click (or hold the control key and click) on the bold text and click 'Inspect'.

The third secret is the word inside the < and > for the tag with the bold text.

Challenge 4

The fourth secret is hidden in the Styles panel. First, find the h1 element - it's the heading of this page, at the top of the <body>, and inside it says 'DevTools Scavenger Hunt'.

Hint: you can scroll to the top of the page, and Inspect the title there.

Click the h1 element in the Elements panel so that the Styles panel shows its styles. The fourth secret is in the content property.

Challenge 5

The fifth secret is hidden in plain sight.

In Elements, click to select the <div> for challenge 5. In the Styles panel, change the background-color from '#F6F6F6' to '#555'.

That will reveal the message hidden below:

the fifth secret is: 'to reach'

Challenge 6

To find the sixth and last secret, you'll have to change another style. There's an image hidden off-screen that has the secret.

Find the <img> tag in the Challenge 6 <div> in the Elements panel.

Click on the <img> in the Elements panel to select it.

In the Styles panel, change the value for 'left' to move the element onto the screen. You can try different values, or delete the 'left' value, that works too!
