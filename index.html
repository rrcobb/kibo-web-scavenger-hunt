1. The first secret message is: Climb Kibo. Type it into the box below.

This one's pretty easy. It's supposed to get you warmed up for solving the next few challenges.

2. To find the second secret message, you'll need to use the developer tools to find the title of the page. Open the developer tools by right clicking anywhere on the page and selecting 'inspect' from the menu. You should see a panel open at the bottom or side of the page. In the Elements tab of the panel, click to open the head tag, and look for the title tag. There, you'll find the secret.
3. The third secret is the name of this HTML element. Use the developer tools again to find the tag's name.
4. The fourth secret is the value of an input element hidden inside this very prompt. Find an input element in this prompt, and look for the value attribute.
5. The fifth secret will be in the Styles tab. First, select the h1 element - it's the heading of the page. The Styles section of the developer tools panel shows the CSS properties and values for this element. The secret is the value for the content property for the h1.
6. To find the last value, you'll have to make a change to the page using the developer tools. There's an image that's positioned off screen. Click on that element in the Elements tab to select it. In the Styles tab, change the value for the 'left' property to move the element so that it's visible.

On the image, you'll see the last secret message.
