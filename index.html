DevTools Scavenger Hunt

Introduction

The DevTools are like a Detective's magnifying glass, notebook, and recorder all in one. Web Developers use the DevTools every day to look 'under the hood' of a webpage, see what's going on, and make experimental changes.

In this challenge, you'll use the DevTools to find messages hidden in the HTML and CSS of this page.

The DevTools

You'll need to use the developer tools to find the messages hidden on this page.

Open the developer tools by right clicking anywhere on the page and selecting 'Inspect' from the menu. You should see a panel open at the bottom or side of the page.

Tour of the devtools

The Elements panel shows the HTML elements that make up the page. You can see what's inside an element by clicking open the arrow on the left.

The Styles panel shows the CSS styles for an element. Click on an element in the Elements panel and look in the Styles panel to see all of the styles.

The Styles panel has a bonus power: you can click on the value for any property and change it! You can change the color of the text, or the font size, or the background color, or any other CSS property.

Tip: You can drag the panels to resize them. There's also a ton more panels to explore later.

Instructions

Click the button below to get started. For each prompt:

  1. Read the clue and use the DevTools to find the answer.
  2. Type the answer in the box.
  3. Click "Check" to check your answer and get the next prompt.

You can check as many times as you want. You'll get feedback on whether your answer was right or wrong.

After you solve all the challenges, you'll uncover the final answer.

Challenge 1

The first secret message is in Elements panel in the <title> inside the <head>. The <title> tag has the secret in it!

In the Elements tab of the panel, click to open the <head> tag, and look for the <title> tag inside.

Type the secret into the box below.

Challenge 2

The second secret is hidden in an input element inside this very prompt.

Click to inspect an element in this paragraph, and look nearby for an <input> element.

The value attribute of the input element contains the second secret.

Challenge 3

The third secret is the name of the HTML tag for this bold text.

Right click on the bold text and click 'Inspect'. The words are inside of a tag, like '<name>'. The tag name is the third secret.

Challenge 4

The fourth secret is hidden in the Styles panel. First, select the h1 element - it's the heading of the page, at the top of the <body>, and inside it says 'DevTools Scavenger Hunt'. Click that element in the Elements panel so that the Styles panel shows all the CSS properties and values. The secret is in the content property.

Challenge 5

The fifth secret is hidden in plain sight.

To reveal it, click to select the <div> for this prompt, Challenge 5. In the styles tab, change the background-color from '#F6F6F6' to '#555'.

That will reveal the message hidden below:

the secret is: 'to reach'

Challenge 6

To find the last value, you'll have to change another style using the developer tools. There's an image hidden off screen that has the secret.

Find the <img> tag by looking near this prompt in the Elements panel. Click on the <img> in the Elements tab to select it.

In the Styles tab, change the value for the 'left' property to move the element onto the screen. You can try different values, there's lots of ones that will work.
